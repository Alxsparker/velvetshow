If you've been using ShowBuddy to manage your backing tracks, you already know what good show control feels like. Velvet Show takes everything that works and adds MIDI automation, OSC support, a real-time iPhone remote and MaestroDMX integration — all in one macOS app.

Why musicians switch from ShowBuddy to Velvet Show

ShowBuddy is a solid tool for managing backing tracks and setlists. But as your show grows more complex — lighting cues, multiple devices, live OSC control — you start running into its limits. Velvet Show is built for exactly that next step.

  • Full MIDI timeline automation — cues fire automatically during playback
  • OSC engine built in — control MaestroDMX, QLab, Companion over WiFi
  • Velvet Remote for iPhone and iPad — real-time two-way sync over local network
  • Lyrics and chords synced to the waveform — not a separate scrolling view
  • Stage screen output for iPad or second monitor
  • PANIC button always one keystroke away
